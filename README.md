# My Projects

## 📊 [Sales Analytics Dashboard](https://github.com/saqib-malik-data-analyst/Sales_Analysis_Dashboard)
This project showcases an interactive **Power BI Sales Analytics Dashboard** that analyzes **Sales, Profit, and Quantity** across the **Central, East, South, and West** regions. The dashboard features dynamic year filtering, metric selection, monthly trends, geographic analysis, and Current Year (CY) vs Previous Year (PY) performance comparison to support data-driven decision-making.
